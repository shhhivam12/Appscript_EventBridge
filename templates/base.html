AppScript Bridge
Quick Start
1
Connect Google Go to Settings → Connect Google Account to authorise the Apps Script REST API.
2
Add a Credential Go to Credentials → Add one for ServiceNow, Telegram, or a custom app. Each credential gets a unique webhook URL.
3
Create a Workspace & Project Workspaces group related projects. Projects hold your workflows.
4
Build a Workflow Inside a project, click New Workflow. Set a Trigger (which credential + event) and an Action (which Apps Script function to call).
5
Activate & Test Toggle the workflow ON, then use the Test panel to fire a sample event and confirm it runs.
Key Concepts
Trigger
An incoming webhook event from an external system (ServiceNow incident, Telegram message, etc.) that starts a workflow run.
Action
The Google Apps Script function that is called when the trigger fires. The event payload is passed as the first parameter.
Credential
Stores authentication details for an external platform and generates a dedicated webhook URL that platform sends events to.
Workspace
Top-level container — like a team or environment. One workspace holds many projects; one project holds many workflows.
